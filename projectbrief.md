# Project Brief

## Назначение проекта
Интернет-магазин UC для PUBG Mobile: лендинг с тарифами, форма ввода Player ID,
корзина, оплата, поддержка.

## Ключевые особенности
- Главная страница показывает список пакетов UC и форму ввода Player ID.
- Корзина и расчёт суммы завязаны на выбранных пакетах UC.
- Оплата идёт через внешнюю платёжную систему (QR НСПК / другое — уточнять у меня).
- Есть FAQ и блоки доверия (PCI DSS, SSL, кол-во клиентов, поддержка в Telegram).

## Где искать основную логику
- Маршруты UI и корзины: routes/web.php.
- Контроллеры покупок/корзины/оплаты: app/Http/Controllers (конкретные классы уточнять у меня).
- Бизнес-логика покупки UC: app/Services (если есть) или внутри контроллеров.
- Шаблон главной страницы UC SHOP: resources/views/... (точный путь я подскажу при необходимости).
- Конфиг платёжки и UC: config/... (подтверждать у меня).

## Окружения
- Local: используется docker-compose и папка docker/ (для разработки).
- Staging и Prod: обычный PHP + nginx, деплой через git pull по SSH, без Docker.
- БД MySQL, размещение и детали подключения уточнять у меня перед серьёзными изменениями.

## Важные правила
- Считать проект продовым: избегать рискованных изменений без плана отката.
- Всегда уточнять:
  - какое окружение затрагиваем (local/staging/prod),
  - затрагиваем ли миграции БД и есть ли бэкап.
